1. (Recommended) Create a python virtual environment and activate it.
2. Run "pip install -r requirements.txt" command to install required libraries.
3. Run "python main.py" to launch program.
(Notice: You can use pycharm instead to create virtual env and run program automatically)
4. After launching program, you can use the below keys:
Press "s" -> run A* algorithm to find battery
Press "b" -> run UCS algorithm to find battery
Press "f" -> run BFS algorithm to find battery
Press "r" -> reset grid
